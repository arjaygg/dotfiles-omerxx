# Goals

Durable, single-outcome goal docs for this project. See `00-index.md` for status.

Filename convention: `YYYY-MM-DD-NN-slug.md`.

Required headings, in order: Objective, Why, Current state, Non-goals, Steps,
Acceptance criteria, Evidence to update, Stop and ask if.

Validate with `python3 scripts/validate_goals.py`.
